- Adding New Addons To Your Project
Any time you want to add new Addons to your verison of OpenFrameworks, you will also need to add those
addons to the addons.mk file in this build directory.

- Using OpenFrameworks Addons In Your Project
In order to use any OpenFrameworks Addons with your project, you will need to create a makefile for that addon. 
You can do this by copying the src.mk file in the root folder of the project and following the instructions within.
